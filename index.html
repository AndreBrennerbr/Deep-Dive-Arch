Por que este projeto existe?

📐

Diagramas, não parágrafos

Conceitos complexos ficam claros quando você vê o dado se movendo. Cada step tem uma visualização Canvas 2D interativa.

🔬

Profundidade real

Sem "é tipo um envelope". Você vai ver o header TCP com seus 20 bytes, o Scaled Dot-Product com a fórmula, o epoll_wait com os flags.

📎

Referências primárias

Cada step aponta para a RFC, o paper original, o código-fonte no GitHub ou a documentação oficial. Nada de "fonte: trust me bro".

🧩

Modular por design

Novos slices podem ser adicionados a qualquer momento. Cada tema é independente — com seus próprios dados e visualizações.
